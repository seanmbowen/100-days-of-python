## 🛠️ Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/YOUR_USERNAME/100-days-of-python.git
   cd 100-days-of-python
   ```

2. **Python version** — Python 3.10+ recommended
   ```bash
   python --version
   ```

3. **Run any project**
   ```bash
   cd day-001
   python main.py
   ```

> Some later-day projects may have additional dependencies. Check the `notes.md` inside each folder for setup instructions.
